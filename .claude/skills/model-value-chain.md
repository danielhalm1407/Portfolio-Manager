## Model Value Chain

When asked to model a segment of the gas/power/electrification value chain:

1. Define the segment boundaries (upstream, midstream, downstream, end-use)
2. Identify key drivers: supply/demand balances, capacity constraints, regulatory factors
3. Map interdependencies with adjacent segments
4. Build a simplified structural model with stated assumptions
5. Output sensitivities: what moves the needle and by how much
